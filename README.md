## Project setup

This project was built with Vite. To run the project, simply clone and run commands:

```bash
## Install packages
  npm install

## Run project on dev enviroment
  npm run dev
```

## My methods

To solitify Javascript fundamental knowledge, I will attempt to build the solution from scratch without following the video tutorial.

Typescript will be ultilised for better type guard, as well as TailwindCSS for styling.
